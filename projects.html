Rainball Web Game


For my final project in my Creative Coding elective last spring, I created an endless scroll web game using a combination of html and javascript. We spent the semester learning the fundamentals of html and javascript, in addition to several javascript libraries that we used to achieve different effects and functionalities on our websites and projects. For this final project, I used a js library called THREE.js to render a 3d game world containing obstacles and the ball which the player controls. I also used another js library to allow my site to interface with an xbox controller for user input.

Excel Report Generation Script


During my internship and part-time employment at Calspan, I developed a Python script to generate PDF reports from excel templates on Linux. The purpose of this project was to simplify the process of making these reports, as the previous method involved running an entire windows VM just to run excel macros. My script uses two different Python APIs to interface with excel files directly on Linux. It uses this functionality to pull in data from separate files to populate an excel template while preserving all charts and style elements, then generating a final PDF from the modifies excel file.
